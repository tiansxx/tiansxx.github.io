使用hexo搭建博客

在GitHub上创建博客

1、在github上创建一个仓库，仓库名为.github.com
2、在新创建的仓库右侧，点settings，进入设置页面，点击Launcher Automic Generate
3、选定想要的网站模版，点publish
4、博客搭建完成，访问http://.github.io就可以访问了

使用Hexo管理

1、安装NodeJS
2、运行npm install -g hexo-cli，安装hexo命令行工具
如果之前改过NodeJS的包默认安装位置，记得在环境变量中配置，否则无法直接在命令行中运行新安装的包命令
3、hexo init <文件夹>
4、cd <文件夹>
5、hexo server
6、访问http://localhost:4000访问当前的博客了

上传

1、修改<文件夹>目录下的_config.yml文件，如下：
deploy:
type: git
repository: git@github.com:/.github.io.git
branch: master
2、部署
hexo clean
hexo generate/g
hexo deploy/d
deploy无法通过，需安装hexo-deployer-git,运行
npm install hexo-deployer-git

Done

写博客

1、命令行运行 : hexo new ‘文章名称’,成功后会生成source/_posts/‘文章名称’.md
2、使用markdown进行编辑
